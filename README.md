# Морской обзор

Ежедневный автоматический обзор происшествий на море: удары БПЛА по судам,
атаки, шторма, действия ВМС России, Украины, Ирана, США.

**Сайт:** https://trainingsw75-cloud.github.io/morskoy-obzor/

Как это работает:

1. Каждый день в **07:40 по Москве** GitHub Actions запускает `build_review.py`.
2. Скрипт читает открытые RSS-ленты YouTube-каналов из `channels.json`,
   отбирает свежие видео по морским и «тревожным» ключевым словам.
3. Строится страница `docs/index.html` (видео встроены плеером YouTube,
   ничего не скачивается и не перезаливается — права остаются у авторов),
   архивная копия в `docs/archive/` и текст `docs/tg.txt` для Telegram.
4. В **08:00 МСК** отдельный робот в репозитории `cook46-vacancy-alerts`
   отправляет `tg.txt` в Telegram через @cook46_bot.

Добавить канал-источник: вписать его в `channels.json` (нужен ID вида `UC…`).
Запустить вручную: вкладка Actions → «Утренний морской обзор» → Run workflow.
